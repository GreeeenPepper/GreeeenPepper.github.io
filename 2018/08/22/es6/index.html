  • let与var的区别

ES5只有全局作用域和函数作用域，没有块级作用域，ES6中新增的let实际上为javascript新增了块级作用域。
let所声明的变量只在let命令所在的代码块内有效。

  • 继承的实现

在ES5中并没有传统的面向对象的概念，javascript利用原型链的特性来实现继承。

顺便再记录一下显示原型prototype和隐式原型prototype的关系：
每一个函数在创建之后都会拥有一个名为prototype的属性，指向函数的原型对象。
任意对象都有一个内置属性，大部分浏览器支持通过proto来访问。
隐式原型指向创建这个函数的constructor的显式原型。

ES6中封装了class，extends等关键字来实现继承。

  • Map + Set + WeakMap + WeakSet 四种集合类型
    四种集合类型，WeakMap、WeakSet作为属性键的对象如果没有别的变量在引用它们，则会被回收释放掉。
